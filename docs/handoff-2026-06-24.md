## Güvenlik düzeltmeleri (bulundu + kapatıldı)

- **Yetki yükseltme (kritik):** `profiles` self-update RLS politikasının WITH CHECK'i
  olmadığından üye, anon key + kendi oturumuyla `PATCH /profiles {role:'admin'}` atıp
  admin olabiliyordu. RLS WITH CHECK OLD/NEW kıyaslayamadığı için BEFORE UPDATE trigger
  (`0021`) ile kapatıldı.
- **Quiz puan manipülasyonu:** `quiz_attempts`'a tüm-kolon INSERT grant'i + yalnız
  user_id'li RLS → üye `{quiz_id,puan:100}` ekleyebiliyordu. INSERT authenticated'tan
  alındı; puanlama yalnız service_role; `submitQuiz` `my_uid()` RPC ile doğrulanmış
  user_id kullanır (`0022`). Kod 0022 öncesi/sonrası uyumlu (canlı bozulmaz).
- **Kabul edilen kalıntılar** (sunucuda doğrulanamaz; README "tehdit modeli"): `lesson_progress`
  self-write (izleme doğrulanamaz, anti-skip best-effort) ve `user_competencies` self-insert
  (kozmetik; puana/liderliğe etki etmez).
